#문제
	이 문제는 문자열에 대한 이해와 활용,
	그리고 map, vecotr, priority_queue에 대한 활용을 다룬 문제입니다.

#풀이
	strtok함수를 이용해 .을 나누어 확장자를 추출.
	추출된 확장자는 키 값으로 map 자료구조에 insert
	만약 같은 확장자가 있을 경우를 대비해 if문으로 조건을 추가함.
	iterator를 이용해 map의 모든 노드를 순서대로 출력.
	이때 map은 key값을 기준으로 오름차순으로 정렬됨.

#난관
	시간이 너무 오래걸린다..
	a[i] = a[i].substr(a[i].find(".") + 1);를 이용해 '.' 이후의 문자열을
	추출하는 방법을 사용할 수 있다.

	확장자를 추출하는 방법을 tok또는 find 로 사용하는 형태로 구현.
	확장자를 저장하는 방법에서 map을 사용하거나 원래 쓰던 배열에서
	갯수를 세는 방식으로 진행하는 경우도 있다.
