#문제
	판을 우선 구현
	4^10가지 경우의 수
	100만가지의 경우의 수..?
	서로다른 말이면 그렇지만
	모두 같은 말이기 때문에 4로 나눠야 할듯.
	같은 말에 대해 
	서로다른 경우의수를 적용하는 경우..?
	모든 경우를 검사하는 브루트 포스이다


#설계
	DFS문제이다
	말이 가는 길은 총 4가지 이다.
	horse에 길과 길 번호를 부여하자.
	갈 수 없는 경우
		현재 위치가 도착지점인 경우
		도착점이 아닌, 진행 위치에 다른 말이 있는 경우

	갈수있다면
		진행한다
		번호가 넘은 경우 번호 조정 해준다.

#난점
	같은 30이라는 숫자인데
	서로 다른 위치에서 만들어진 30인경우
	다르게 인식해야하는데 그게 안됫다.
	dicego 부분에서 수정함.

	//잠시 해설을 보류한다
	값이 같거나
	길이 같아도
	서로 다른 점일수도
	같은 점일 수도 있다.
	길을 4개로 만들어 놨기에
	서로 다른 길인경우,
	하나하나 따져줘야한다.

#풀이
	다른 사람의 풀이를 봤을때 훨씬 간단했다
	맵 하나를 이용해 표시를 했으며
	같은 숫자에 대해 코너를 밟고 갔느냐 안갔느냐로 구분하고
	30과 40에 대해서만 특별한 조건을 추가하면
	충돌조건은 완성하기 쉬웠다.
	기본은 숫자를 이용해 같은지 다른지를 판단하는점,
	코너를 돌았는지 안돌았는지를 이용해 판단하는점.
	너무 간단한 원리를 왜 이렇게 어렵게 생각했을까 ㅠㅠ
